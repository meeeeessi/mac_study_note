### Linux命令

cat /etc/passwd|grep mysql

cat /etc/group|grep mysql

开启mysql：systemctl start mysqld.service

关闭mysql：systemctl stop mysqld.service

查看mysql是否启动：ps -ef|grep mysql

设置mysql密码：/usr/bin/mysqladmin -u 用户名 password 密码
